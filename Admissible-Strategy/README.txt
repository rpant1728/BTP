Dependencies - 

1. A C++ Compiler
2. Python 3.x
3. pg-solver (follow instructions given in pgsolver-master/doc/pgsolver.pdf, section 3.3, for installation)

To run program on a random input -

    bash run.sh GRAPH_SIZE

To get average runtimes of optimized and unoptimized programs - 

    bash get_runtimes.sh GRAPH_SIZE NUM_OF_ITERATIONS

To visualize the random input graph -

    bash visualize.sh
